## Requerimentos

### Node

- Node >= 12

### Yarn

- Versão = latest

## Instalação

Clonar o Repositório e importar projeto no VSCode

```sh
$ git clone
```

## Como rodar o app

```sh
$ yarn install ou npm install
```
### ANDROID
```sh
$ yarn start
$ yarn android
```

### IOS
```sh
$ npm install
$ cd ios
$ pod install
$ cd ..
$ yarn ios
```

## Como tirar os warnigs no app

Colocar a linha no index.js console.disableYellowBox = true;
